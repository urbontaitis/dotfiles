## ...
The hyper key is defined as ⌘ + ⌥ + ⌃ (ctrl + alt + cmd). Press this plus the defined key:

Key | Action
---|---
A | Lock Screen
R | Reload config
W | Print `¯\_(ツ)_/¯`

## Launch Apps

Key | App
---|---
B | Firefox
E | Visual Studio Code
F | Finder
M | Mail
P | System Preferences
S | Spotify
T | iTerm

## Window Management

Key | Action
---|---
c | Move window to the center
← | Move window left
→ | Move window right
↑ | Move window up
↓ | Move window down
i | Decrease height
k | Increase height
j | Decrease width
l | Increase width
2 | Set 2x2 grid
3 | Set 3x3 grid
4 | Set 4x4 grid
/ | To next screen
, | Snap to grid
(space) | Maximize window
. | Minimize window
